 # Apresentações das Entregas
 
 ## Introdução

 Esta página reúne, de forma sintética e organizada, as apresentações realizadas no âmbito do projeto da disciplina Requisitos de Software (2025.2). Para cada etapa são indicados os objetivos, os participantes, os artefatos produzidos e os links para gravações ou documentos de verificação, de modo a facilitar a consulta às evidências e acompanhar a evolução do trabalho.
